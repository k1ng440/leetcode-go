# Boilerplate Generator
generates boilerplate from given URL.

## Files
* README.md - problem description
* *problem name*.go - code snippet provided by leetcode


## Usages
```
go run main.go URL
go run main.go https://leetcode.com/problems/reverse-linked-list/
```
